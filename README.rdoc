== README

You can download this application and just boot it from your local directory if you have ruby on rails installed. In blog directory just type "rails s" to boot up the blog.

* Ruby version - ruby '2.1.8'

* Database - 'PostgreSQL' 

* Gem list - Available in gemfile


*TO ADD USERS TO ALLOW FOR POSTING, YOU MUST ENABLE THE REGISTRATION IN model/user.rb
